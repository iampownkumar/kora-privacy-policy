Kora Launcher – Privacy Policy

Effective Date: March 24, 2026

Kora Launcher is designed to help users reduce screen time and use their devices more intentionally. Your privacy is a core principle of this app.

1. Data Collection and Usage

Kora Launcher does not collect, store, or transmit personal data to any external servers.

Permissions Used:

  • Usage Access: Used to track app usage for core functionality.
  • Default Launcher: Required to function as a launcher.
  • Local Storage: Stores data only on your device.

2. Data Sharing

No data is shared, sold, or transmitted.

3. User Control

All data can be deleted by uninstalling the app.
